What is CSS?

Definition

CSS, also known as Cascading Style Sheets is a stylesheet language mainly or knowledgeably used for, as the name suggests, used for styling.
It is a stylesheet language used to control the presentation and layout of HTML Documents. To achieve visually appealing and responsive web pages.

What have we learned in CSS?

Given from the definition, we have learned a lot of styling, layouts and designing in CSS to create an attractive layout for the HTML website.
From our experience, there is nothing much towards CSS other than said styling.
